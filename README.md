# Greater Uni Hackathon 2025

Collection of challenge solutions and prototype applications built for the Greater Uni Hackathon 2025. Each folder contains source code, writeups, and supporting assets for a specific track or prompt.

## Projects
- `Hackster_GUH` – Security-focused reverse engineering and jail escape writeups. Includes the `PyJail` sandbox challenge and supporting documentation.
- `Housr_GUH` – Frontend marketplace dashboard built with React, Vite, Tailwind CSS, and shadcn/ui to showcase premium home furnishings.
- `Reply_GUH` – Emergency response coordination platform with multi-role interfaces and Supabase edge functions for AI-assisted triage.
- `Quant_GUH` – Quantitative trading starter bot and credentials scaffolding for the Limex market-making challenge.

## Getting Started
```sh
git clone https://github.com/jonghunlee05/Greater_Uni_Hackathon_2025.git
cd Greater_Uni_Hackathon_2025
```

Each sub-project maintains its own tooling. Refer to the project-level `README.md` files for installation and run instructions.
